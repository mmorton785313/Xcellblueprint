# Awaken & Reign Studios Website

## Fast launch
1. Create a GitHub repository.
2. Upload `index.html`, `styles.css`, and the `assets` folder.
3. Go to Vercel.com and import the GitHub repository.
4. Click Deploy.

## Update later
- Replace any image in `assets` with a new image using the same filename.
- In `index.html`, replace the Facebook links with your real Messenger link.
